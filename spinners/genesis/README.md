# Genesis Spinner

Provisions a Webspinner Cell from a fresh macOS or Linux host. The encoded version of the founding bootstrap.

| File | What it is |
|---|---|
| `manifest.json` | Capability declarations — eight steps from toolchain to verified Cell. |
| `how-it-works.md` | Operative explanation of every step, rendered in the Loom. |
| `src/index.ts` | Entrypoint contract (open work — handlers not yet implemented). |

The Genesis Spinner is intentionally not yet runnable: its handlers are next-turn work. The recipe it encodes — the steps Claude Code ran by hand once on Kepler — is in `how-it-works.md`. See `DECISIONS.md` 2026-05-10 — *Genesis Spinner: encoding the founding bootstrap*.
